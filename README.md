#### 环境搭建

    ```
    python 2.7.6+
    pip install uiautomator
    ```


### 接口

    * 更新HTML文件
        ```python
        python manage.py updatehtml /file/path/htmldir
        ```

    * 把HTML转换成XML
        ```python
        python manage.py htmltoxml /file/path/htmldir/html.json
        ```
